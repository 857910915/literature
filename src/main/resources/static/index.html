采桑子.秋荷        采桑子.秋荷          李知了西风斜柳摇波影，飞絮游丝。雾绕烟迷，袅袅凝霜锁瘦池。娇黄莲叶生寒露，凋碧芳菲。多少相思？留待枯枝泣别离。  
七律·秋近 神清气朗天高处，舒卷无形又一秋。半点新黄随季上，万千墨绿就风柔。晨来剪破霜成梦，夜落铺张夕染舟。渐醉风声翻册急，残荷雨径可知不。
南乡子 秋花 陌上泛红英，虽错三春也可倾。寒露湿来花更艳，层层，多少相思叠得成。何意苦相争，已是西风不解情。暮里且开晨且落，谁凭，昨夜星辰枉自听。
